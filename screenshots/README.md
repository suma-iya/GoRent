# Screenshots Directory

Place all your screenshot images in this folder, then run the renaming script from the project root:

```bash
./rename_screenshots.sh
```

## Expected Screenshot Files

After renaming, your screenshots should have these names:

### Authentication & Registration
- `login.png` - Login screen
- `register.png` - Registration screen

### Property Management
- `properties.png` - Managed properties list
- `add_property.png` - Add property dialog
- `property_details.png` - Property details view
- `add_floor.png` - Add floor dialog
- `update_floor.png` - Update floor dialog
- `tenant_request.png` - Send tenant request dialog

### Payment Management
- `payment_details.png` - Payment details screen
- `payment_history.png` - Payment history table
- `send_payment.png` - Send payment dialog
- `adjust_rent.png` - Adjust due rent dialog
- `advance_request.png` - Advance payment request
- `advance_details.png` - Advance payment details

### AI Chatbot
- `chat.png` - Chat interface
- `risk_analysis.png` - Risk analysis response
- `monthly_summary.png` - Monthly risk summary
- `recommendations.png` - Action recommendations
- `high_risk.png` - High risk tenants list
- `compare.png` - Tenant comparison

### Notifications
- `notifications.png` - Notifications list
- `notification_actions.png` - Notification with action buttons
- `notification_accepted.png` - Accepted notification

### Settings & Localization
- `settings.png` - Settings screen
- `language.png` - Language selection dialog
- `bengali_ui.png` - Bengali language UI example

## Manual Renaming

If the script doesn't work or you prefer to rename manually, use the filenames listed above. The README.md in the project root references these exact filenames.
